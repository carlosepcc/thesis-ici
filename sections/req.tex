\subsection{Requisitos no funcionales}

\noindent Usabilidad:
\begin{RNF}
\item El sistema debe ser una aplicación web
\item La interfaz del sistema debe ser intuitiva y fácil de usar
\end{RNF}
Hardware y software:
\begin{RNF}[resume]
\item El sistema operativo del servidor debe tener instalado la máquina virtual de Java 
\item El ordenador donde se ejecute el servidor debe poseer como mínimo 4 GB de RAM y 10 GB de almacenamiento
\item El cliente del sistema se podrá ejecutar en cualquier sistema operativo que posea un navegador web
\end{RNF}
Seguridad:1
\begin{RNF}[resume]
\item Solo se podrá acceder al sistema después de estar autenticado con usuario y contraseña válidos
\item Solo se podrán usar las funcionalidades del sistema en dependencia de si el usuario autenticado posee los permisos requeridos para la misma
\end{RNF}
Diseño e implementación:
\begin{RNF}[resume]
\item Para el desarrollo del lado del cliente se usará VueJS
\item Para el desarrollo del lado del servidor se usará Java con SpringBoot
\item Usar la metodología de desarrollo de software \ac{XP}
\end{RNF}
