\section{Tareas de Ingeniería}
La metodología de software XP plantea que la implementación de un software se hace iterativamente.
Durante cada iteración se desarrollan un conjunto de HU definidas por el cliente y descritas por el equipo de desarrollo. En esta fase de implementación las HU se dividen en tareas de ingeniería, las cuales son asignadas a los programadores para ser implementadas durante la iteración correspondiente \citep{Joskowicz2008}.\\
A continuación, se muestran algunas de las tareas de ingeniería:
